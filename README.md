# FolkPatch 主题概述

如你所见这就是一个主题的仓库


反正就是FolkPatch的第三方主题

### ✨ 主要特性
- **简洁**：一眼望去非常简洁
- **易于使用**：简单直观的操作方式
- **可导入**：支持导入到FolkPatch主题

## 📋 系统要求

### 基本要求
- **操作系统**：[安卓8-16+]
- **运行环境**：[FolkPatch1.0+]


---

## 📥 安装指南

### 方法一：直接应用（推荐）
1. 下载 `.fpt` 主题文件
2. 进入手机 **FolkPatch**
3. 点击 **“设置” → “外观” → “导入主题”**
4. 选择下载的主题文件并应用

### 方法二：手动安装
1. 将主题文件复制到：
   ```
   /data/user/0/me.yuki.folk/files/    
   ```
   图片需要自行寻找替换对应名字
   
3. 在APP中刷新即可显示本地主题
